# KY-131 Announcements Editor

A minimal web app for editing `html.html` via a GUI. Paste your raw announcements, parse, edit, preview, and export the final HTML. Designed for GitHub Pages deployment.

## Usage
- Paste your announcements text
- Click "Parse & Preview"
- Edit each section in the form
- Preview the result live
- Click "Export HTML" to download the updated `html.html`

## Development
- `npm install`
- `npm run dev` (for local dev)
- `npm run build` (for production build)

## Deployment
Deploy the `dist/` folder to GitHub Pages.
